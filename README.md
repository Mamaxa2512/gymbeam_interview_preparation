# gymbeam_interview_preparation

## Як запустити скрипт

Щоб запустити скрипт `recomended_price.py`, відкрийте термінал у папці з проєктом та виконайте команду:

```bash
python3 recomended_price.py
```

Переконайтеся, що у вас встановлена бібліотека `pandas`. Якщо ні, встановіть її командою:

```bash
pip install pandas matplotlib
```

## Що робить скрипт:
1. Завантажує дані з `internal_sales.csv` та `competitor_prices.csv`.
2. Розраховує рекомендовані ціни за бізнес-правилами (зберігаючи маржу ≥ 15%).
3. Виводить фінансовий звіт у термінал.
4. Зберігає детальну таблицю у файл `recommended_prices_output.csv`.
5. Генерує 3 аналітичні графіки у форматі `.png`.
